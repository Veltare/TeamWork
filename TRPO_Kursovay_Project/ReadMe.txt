Это приложение TRPO_Kursovay_Project создано автоматически с помощью мастера приложений.

В этом файле представлена сводка содержимого всех файлов, входящих в состав приложения TRPO_Kursovay_Project.


TRPO_Kursovay_Project.vcxproj
    Это основной файл проекта VC++, создаваемый с помощью мастера приложений. Он содержит данные о версии языка Visual C++, использованной для создания файла, а также сведения о платформах, конфигурациях и функциях проекта, выбранных с помощью мастера приложений.

TRPO_Kursovay_Project.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера приложений. Он содержит сведения о сопоставлениях между файлами в вашем проекте и фильтрами. Эти сопоставления используются в среде IDE для группировки файлов с одинаковыми расширениями в одном узле (например CPP-файлы сопоставляются с фильтром "Исходные файлы").

TRPO_Kursovay_Project.cpp
    Это основной исходный файл приложения.

/////////////////////////////////////////////////////////////////////////////
Кроссворд - это игра для проверки знаний и эрудиции.
Игра представляет из себя консольное приложение. 
Во время игры пользователю будет предоставлен выбор плоскости ответов(по горизонтали/вертикали).
Вводимые ответы будут отображаться в консольном окне.Игра предусматривает 3 уровня сложности.(1.Легкий.2.Средний.3.Сложный).Переход на новый уровень будет осуществляться по завершению ответов на все вопросы.Игра не предоставляет выбор уровня сложности,переход осуществляется по иерархической лестницы(от легкого к сложному).
Тематика вопросов затрагивает основные парадигмы и фундаментальные знания в области информационных технологий.Количество вопросов зависит от диапазона случайных значений.Легкий(7-12)/Средний(12-16)/Сложный(16-20).
